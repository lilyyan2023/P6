Safe On Campus
In 2020 summer, I and my team attended the Unicode SC+ hackathon and we needed to design a solution for ensuring in-person socializing safety during COVID19 within 36 hours. I was responsible of using React and JavaScript to create a website to allow users to report areas around Northwestern campus for any unsafe behavior or large gathering. The integrated Google map will directly reflect the number of reports using color gradient. We also require user to update their daily symptoms using form we created and based on their symptoms, we can provide green/red health status bar for users to show when entering certain buildings.
